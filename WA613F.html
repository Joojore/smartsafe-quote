WA613F vs WA613: В ЧЁМ РАЗНИЦА?
WA613F — напольная версия для смотровых ям. Блоки ставятся на пол по бокам ямы. Не нужен подъёмник. Идеально для гаражей с ямой вместо подъёмника.

WA613 — стандартная версия для подъёмников. Блоки крепятся магнитами на стойки подъёмника. Для цехов с двухстоечным/четырёхстоечным подъёмником.

Технические характеристики ОДИНАКОВЫ. Разница только в способе установки. Оба поддерживают 4-точечный ступичный и 3-точечный шинный зажимы.

Тележки для зажимов: мобильные стойки на колёсах для хранения и перемещения зажимных устройств между постами. Удобно, когда в цехе несколько подъёмников/ям.
